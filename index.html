Fichiers proposés en téléchargement par Planète Excel

Des fichiers Excel prêts à l'emploi, proposés par www.planete-excel.fr, le site dédié à Excel :

Liste des communes de France 2023, avec coordonnées de géolocalisation

Le fichier est une agrégation des données trouvées sur le Code officiel géographique au 1er janvier 2023 (Insee) et sur data.gouv.fr.

Liste des colonnes :

  • Nom (majuscules),
  • Nom (minuscules),
  • Nom avec article,
  • Code commune,
  • Code commune parent,
  • SIREN,
  • Code région,
  • Région,
  • Code département,
  • Département,
  • Code Canton,
  • Canton,
  • Arrondissement,
  • Adresse,
  • Code postal,
  • Bureau distributeur,
  • Coordonnées GPS de la mairie (Longitude, Latitude).
